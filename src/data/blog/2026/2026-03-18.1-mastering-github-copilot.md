# Mastering GitHub Copilot

🤖 GitHub: <a href="https://github.com/microsoft/Mastering-GitHub-Copilot-for-Paired-Programming">Mastering-GitHub-Copilot-for-Paired-Programming</a>

Microsoft have invested in teaching people to use GitHub Copilot with this repo that has a number of lessons and exercises via codespaces.

IMO, this is a Good Thing (tm)! Sure, this "free resource" is designed to make you spend money buying GitHub Copilot subscriptions. _I totally get that/they are a business_. But it's a great resource for learning how to use GitHub Copilot and how to get the most out of it. It's also a great way to learn how to use AI-assisted development in general.

You just need to create a GitHub account, sign up for GitHub Copilot (there's a free trial) and then you can start learning.

I'm off to do this myself. Even though I know some stuff, I'm sure it's just the basics.

Finally - I also appreciate that there's a heap of anti Copilot sentiment out there. I'm not experienced with other IDE's like OpenCode or Claude Code (because I cannot afford those plans, even though they are not too expensive) but I do have a paid version of Copilot so I'll be happy with what I have access to.
